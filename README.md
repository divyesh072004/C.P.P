# Banking System Project

## Features
- User Registration
- Account Management
- Fund Transfer
- Transaction History
- Loan Management

## Compilation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/divyesh072004/c-programming.git
   ```
2. Navigate to the project directory:
   ```bash
   cd c-programming
   ```
3. Compile the source code:
   ```bash
   gcc banking_system.c -o banking_system
   ```

## Usage Guide
- Run the program:
   ```bash
   ./banking_system
   ```
- Follow the on-screen instructions to use the system.
